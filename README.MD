xpell-vibe-starter

Real-time Vibe Coding starter for Xpell 2.

Modify your app while it’s running — no rebuild.

⸻

What it shows
	•	Live app loaded from server
	•	Real-time UI mutation
	•	Preview → Apply → Persist
	•	Wormholes via xnode
	•	server-xvm persistence

⸻

What this repository is NOT

This repo does NOT include:
	•	multi-tenant SaaS logic
	•	chatbot / agent flows
	•	onboarding systems
	•	voice features
	•	admin product logic

Core idea

Most AI tools rebuild apps.

Xpell updates a running app.
	•	No rebuild
	•	No reload
	•	Minimal tokens
	•	Instant feedback

⸻

Goal

This starter demonstrates:

prompt → preview → apply → persist

in a real Xpell runtime environment.
